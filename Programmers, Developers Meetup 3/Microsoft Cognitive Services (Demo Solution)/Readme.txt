Prerequisites:
	1. Extract the content of the zip file.
	2. Open the Solution file.
	3. Restore Nuget Packages and Build the Solution.
	4. Put your Emotion, Face and Computer Vision API keys on Constants.cs file under the PDCognitive folder.
	5. At this point, you're good to go. Try running the application on an actual Android Device.
